Adds a temporal phase portrait to the chart with the given parameters.

Parameters
plottitlethe title of the plot, which will be displayed in the legend.
lineStyledescribes how the line used to connect the data points should look, as described in LineStyle
symbolStyledescribes how the symbol used to represent data points should look, as described in SymbolStyle
memoryDepthThe number of data points that will be remembered by the plot at one time. If a value of -1 is given, this will be auto-calculated based on the memory depth of the chart and the data rate.
Returns
A pointer to the constructed temporal phase portrait object.
