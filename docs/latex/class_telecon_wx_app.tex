\doxysubsection{Detailed Description}
Main GUI for telecon object, inherits from wx\+App. Holds references to all subsections of the app, such as Telecon\+Wx\+Windows.

As opposed to \mbox{\hyperlink{class_telecon}{Telecon}}, this object is an {\itshape implementation} of the data described in \mbox{\hyperlink{class_telecon}{Telecon}} and related objects. It is created automatically by \mbox{\hyperlink{class_telecon}{Telecon}}, and should not be interacted with directly by the user.

This class, and all classes starting with the prefix Telecon\+Wx, are wx\+Chart\+Dir-\/based implementations of charts and plots. In the future, \mbox{\hyperlink{class_telecon}{Telecon}} may support other implementations using other frameworks, which would use the same user-\/level objects to interact (such as \mbox{\hyperlink{class_telecon}{Telecon}}, \mbox{\hyperlink{class_telecon_window}{Telecon\+Window}}, \mbox{\hyperlink{class_telecon_chart}{Telecon\+Chart}}, and \mbox{\hyperlink{class_telecon_plot}{Telecon\+Plot}}).
