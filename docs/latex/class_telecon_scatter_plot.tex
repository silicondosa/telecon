\doxysubsection{Detailed Description}
The user-\/level interface for interacting with scater plots. A scatter plot is a sequence of 2-\/dimensional points that consist of a timestamp, displayed on the x-\/axis, and a value, displayed on the y-\/axis. The points should be represented with a symbol. Similar to a \mbox{\hyperlink{class_telecon_line_plot}{Telecon\+Line\+Plot}}, but with no line connecting data points.
