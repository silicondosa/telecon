Adds a chart to the window. Charts will be displayed in the order added, from top to bottom.

Parameters
titlethe title displayed above the chart.
ylabelthe label displayed adjacent to the y-axis.
xlabelthe label displayed beneath the x-axis.
dataIntervalthe expected time, in milliseconds, between data pushes. This will only affect the area of the plot displayed, and will not cause an error if the actual time between pushes differs from this value.
memoryDepththe maximum number of data points that will be held in the data buffer before old values start to be dropped.
colorSequenceModethe sequence of colors that will be used for added plots if the color is left unspecified.
Returns
A pointer to the TeleconRealTimeChart object created.
