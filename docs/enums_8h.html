enum PlotColor

Describes what color a line or symbol will have. In general, colors are represented as RGB colors, with one byte for each channel. These enumerators are just aliases for the colors they represent. This enum also provides the special color COLOR_DEFAULT, documented below.

Enumerator
COLOR_DEFAULT 

This color has special behavior depending on the context in which it is used, according to the following rules (in order of priority):

  1. If used for a plot's SymbolStyle, and that plot has a LineStyle, the symbol color will match the line color.
  2. If used for a plot in a chart's add plot method, will use the color indicated by the chart's ColorSequenceMode.
  3. Otherwise, defaults to black.

enum PlotSymbol

Describes the symbol that a plot will place to represent each data point.

Enumerator
SYMBOL_NO_SYMBOL 

Does not use a symbol; data points are left unmarked.

SYMBOL_CIRCLE 

Marks each data point with a circle.

SYMBOL_SQUARE 

Marks each data point with a square.

SYMBOL_TRIANGLE 

Marks each data point with a triangle.

SYMBOL_DIAMOND 

Marks each data point with a diamond (a square rotated 45 degrees).

SYMBOL_CROSS 

Marks each data point with a cross (a times sign, or a plus sign rotated 45 degrees).

SYMBOL_PLUS 

Marks each data point with a plus sign.

SYMBOL_STAR 

Marks each data point with a five-pointed star.
