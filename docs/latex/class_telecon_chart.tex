\doxysubsection{Detailed Description}
The user-\/level chart interface. It describes features such as the title, data rate, x/y labels, and color sequence that will be used.

This class is abstract. A subclass such as \mbox{\hyperlink{class_telecon_realtime_chart}{Telecon\+Realtime\+Chart}} or \mbox{\hyperlink{class_telecon_data_chart}{Telecon\+Data\+Chart}} should be used to add plots.
