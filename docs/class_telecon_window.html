Adds a chart to the window. Charts will be displayed in the order added, from top to bottom.

Parameters
titlethe title displayed above the chart.
memoryDepthSecondsthe default number of seconds for which to store data. This can be overridden on individual plots, but may result in odd spacing if non-default values are used.
dataRateMillisthe rate at which data will be provided to the plots via pushData functions. If the actual data rate differs from the value provided here, it will result in spacing issues but no fatal errors.
xLabelthe label displayed beneath the x-axis.
yLabelthe label displayed adjacent to the y-axis.
colorSequenceModeDescribes how plots using the default color will be colored. For a full description, see TeleconChart::TeleconChart.
Returns
A pointer to the TeleconRealTimeChart object created.
